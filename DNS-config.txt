# DNS Configure on Cisco Router

Topology
========
                                                                         PC1
    8.8.8.8                                   g0/0     g0/1              |.101           
{dns.google.com}--------------{Internet}-----------(+)------------------[=]----.102-PC2
                                         .1   .192  R1 .1                |.103
                                    192.12.122.0/24      192.168.0.0/24  PC3

Configuration on R1
===================
config t
ip dns server
ip host R1 192.168.0.1
ip host PC1 192.168.0.101
ip host PC2 192.168.0.102
ip host PC3 192.168.0.103
ip name-server 8.8.8.8
ip domain lookup
exit
show host
